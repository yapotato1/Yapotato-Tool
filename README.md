# 抖音无水印视频下载工具

一个简单易用的抖音视频下载工具，支持下载无水印视频。完全本地运行，不依赖第三方服务，保护您的隐私。

## ✨ 特性

- 🎯 **简单易用**：粘贴链接即可下载
- 🚫 **无水印**：下载的视频不含抖音水印
- 🔒 **隐私安全**：完全本地运行，不上传任何数据
- 🌐 **跨平台**：支持 Windows、macOS、Linux
- 📱 **响应式**：支持桌面和移动浏览器
- ⚡ **快速**：直接从抖音官方接口获取视频

## 📋 系统要求

- Python 3.7 或更高版本
- 现代浏览器（Chrome、Firefox、Edge、Safari）

## 🚀 快速开始

### 1. 安装依赖

首次运行时，程序会自动安装所需依赖。如果需要手动安装：

```bash
pip install flask flask-cors requests
```

### 2. 启动服务器

```bash
python server.py
```

服务器将自动启动并在浏览器中打开 `http://127.0.0.1:8888`

### 3. 使用方法

1. 在抖音 App 中找到想要下载的视频
2. 点击"分享"按钮，选择"复制链接"
3. 将链接粘贴到工具的输入框中
4. 点击"解析"按钮
5. 预览视频后，点击"下载"按钮
6. 视频将自动下载到浏览器的默认下载文件夹

## 📖 支持的链接格式

- 标准链接：`https://www.douyin.com/video/7123456789012345678`
- 短链接：`https://v.douyin.com/xxxxxx/`
- 分享链接：`https://www.iesdouyin.com/share/video/7123456789012345678`

## 🛠️ 技术架构

### 后端
- **Flask**：轻量级 Web 框架
- **Requests**：HTTP 请求库
- **Python 标准库**：正则表达式、日志等

### 前端
- **HTML5 + CSS3**：现代化界面
- **原生 JavaScript**：无需额外框架
- **Fetch API**：异步请求

### 工作原理
1. 解析用户提供的抖音链接，提取视频 ID
2. 通过抖音官方 API 获取视频元数据
3. 提取无水印视频 URL
4. 触发浏览器下载

## ❓ 常见问题

### 1. 服务器无法启动

**问题**：运行 `python server.py` 后报错

**解决方案**：
- 确保已安装 Python 3.7+：`python --version`
- 手动安装依赖：`pip install flask flask-cors requests`
- 检查端口是否被占用：修改 `PORT` 配置

### 2. 解析失败

**问题**：提示"无法从链接中提取视频ID"

**解决方案**：
- 确保链接是完整的抖音视频链接
- 支持的格式：`douyin.com`、`v.douyin.com`、`iesdouyin.com`
- 检查网络连接是否正常

### 3. 视频下载失败

**问题**：解析成功但下载失败

**解决方案**：
- 检查浏览器是否阻止了下载
- 查看浏览器的下载设置
- 确保有足够的磁盘空间
- 某些视频可能是私密视频或已被删除

### 4. 视频仍有水印

**问题**：下载的视频仍然有水印

**解决方案**：
- 这种情况很少见，可能是抖音 API 返回的就是带水印版本
- 尝试重新解析链接
- 某些特殊视频可能无法获取无水印版本

### 5. 网络错误

**问题**：提示"网络连接失败"

**解决方案**：
- 检查网络连接
- 确认可以访问抖音网站
- 检查防火墙设置
- 某些网络环境可能限制访问
